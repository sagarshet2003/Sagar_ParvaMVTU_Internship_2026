Github commands:
=================
1. git init --> to initialize the git repo in cwd.
2. git clone "url" --> to clone the remote repo in local
3. git status --> to check the status of repo.
3. git add . --> to stage the files in cwd.
4. git commit -m "message" --> to commit changes
5. git push origin main --> to push the code from local to remote.
6. git pull origin main --> to pull the code from remote to local.
7. git config --global user.name "user name" -- to configure the credentials in global
8. git config --global user.email "email" --> to configure the credentials in global



To add something use follwing instructions
1.git status
2. git add .
3. git commit -m "updated index.html"
4. git push origin main
