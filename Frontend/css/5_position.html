Position property:

used to control the position of html elements

        1.position:static - this is default position of all html elements(we cannot use top , left, bottom and right properties with this position)
        2.position:relative - this will position the element from original position
        3.position : absolute - this will position the element from nearest positioned ancestor
        4.position:fixed - it will fix the html element
        5.position:sticky - it will stick the html element when we scroll the page
